Internship Day 3 Documentation

Day 3 Overview

On Day 3 of the internship, I explored 3D design and modeling workflows using Autodesk Fusion and Blender. I learned the fundamentals of CAD modeling and sculpting techniques used in product design and digital modeling.

Autodesk Fusion Learning

  • Learned the basics of Autodesk Fusion interface and tools
  • Understood sketching and 3D extrusion workflow
  • Created a 3D model of a mug using CAD techniques
  • Learned basic design workflow used in product modeling

Fusion 3D Mug Model

Designed and modeled a basic 3D mug using Autodesk Fusion. Learned sketching, extrusion, hollow body creation, and handle modeling techniques.

Blender Sculpting

  • Explored Blender sculpting workspace
  • Learned basic sculpting brushes and tools
  • Understood mesh manipulation concepts
  • Practiced digital sculpting workflow

Blender Sculpting Practice

Practiced digital sculpting in Blender using sculpt mode tools. Explored face shaping, mesh manipulation, and basic sculpting workflow.

Learning Outcome

Day 3 improved my understanding of CAD modeling, digital sculpting, and 3D design workflows. I learned how engineering design software and creative tools are used in product development and visualization.
